# 📖 About

SigNova is an AI-powered communication platform designed to bridge the communication gap between **deaf** and **non-deaf** people through intelligent Sign Language translation.

The platform provides a seamless communication experience using Artificial Intelligence and an accessible cross-platform mobile application.

---

# ✨ Features

✅ 🤟 Text → Sign Language Translation

✅ 💬 Sign Language → Text Translation

✅ 🤖 AI-powered Chat Translation

✅ 🔐 Secure User Authentication

✅ 📜 Translation History

✅ 📱 Cross-platform Mobile Application
